# Battle of Baytaro

A cyberpunk-themed typing battle game where you help Minataro defeat enemies by typing words correctly.

## Game Modes

- **Hiragana Mode** - Type the romaji for displayed hiragana words (55 words, levels 1-4)
- **English Mode** - Spell displayed English words correctly (100 words, levels 1-5)

## Features

- Battle system with HP, combos, and special moves
- 5 stages with increasing difficulty
- Adaptive word selection that targets your weak spots
- Daily and all-time record tracking (localStorage)
- Procedural sound effects (Web Audio API)
- Neon/cyberpunk visual theme with animations

## Tech Stack

- React 19 + TypeScript
- Vite
- Pure CSS (no UI libraries)

## Development

```bash
npm install
npm run dev
```
